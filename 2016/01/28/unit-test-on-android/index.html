总结

  • android.jar 是被 mock 过的，所以有些 Android API 返回值可能会不是预期值。
  • JUnit 本身是针对 java 代码的单测，所以尽量不要用于使用了 Android API 的方法。
  • Android Gradle 插件提供的 mocked android.jar 跟我刚开始工作时做过的测试框架原理类似。
