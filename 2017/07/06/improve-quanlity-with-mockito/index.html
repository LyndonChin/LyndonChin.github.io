Mockito 用法入门

Mockito是一个用于Java单测的Mock框架，它可以帮我们改变类或者对象的行为。
Mockito能够让我们可以更专注于去测试“代码单元”的逻辑，而无需花费心思去构造数据。

Mock可分为两种类型，一种是Class Mock，另一种是Object Mock（Mockito叫spy）。
改变mock对象方法（method）叫Stub（插桩）。
一次Mock过程称为Mock Session，基本包含如下三个步骤：

+-------------+      +------+      +--------+
| Mock or Spy | ===> | Stub | ===> | Verify |
+-------------+      +------+      +--------+

Class Mock

Class Mock改变了class的行为，所以mock出来的对象就失去了原来的行为如果没有对method进行插桩，那么会返回默认值（null、false、0等）。
