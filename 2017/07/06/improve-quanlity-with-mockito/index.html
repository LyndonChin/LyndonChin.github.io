Mockito 用法和原理

Mockito 是一个用于 Java 单测的 Mock 框架，它可以帮改变一个类或者对象的行为。
Mockito 能够让开发者更加专注地去测试代码单元的逻辑，而无需花费心思构造数据。

Mock 可分为两种类型，一种是 Class Mock，另一种是 Object Mock（Mockito 叫 spy）。
改变 mock 对象方法（method）的行为叫 Stub（插桩）。
一次 Mock 过程称为 Mock Session，基本包含如下三个步骤：

+----------+      +------+      +--------+
| Mock/Spy | ===> | Stub | ===> | Verify |
+----------+      +------+      +--------+
